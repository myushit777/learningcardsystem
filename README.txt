**Learningcardsystem - README**

Hier sind einige wichtige Informationen zum Learningcardsystem:

1. **Postman Collection**: In der Datei "Postman_Collection.json" finden Sie alle POSTMAN-Befehle für die API.

2. **Controller**: Dieses Projekt verwendet nur einen Controller.

3. **Services**: Es gibt zwei Services, einen für Karten und einen für Kartensets.

4. **CRUD-Methoden**: Beide, Karten und Kartensets, bieten CRUD-Methoden an.

5. **Antworten**: Verwenden Sie "userAnswer" für alle Antworten, da Eingaben immer als Strings erwartet werden. Es gibt separate Antwortbefehle in Postman für den Umgang mit Multiple-Choice-Fragen, die mehrere Antworten in einem kommagetrennten String akzeptieren.
